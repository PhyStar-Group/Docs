## 引导

1. **快速开始**：快速运行程序，实现第一个例子。
2. **新手指南**：针对不了解或不熟悉Linux系统以及开发环境的使用者，给出一些使用建议
3. **使用手册**：程序具体的使用手册与指南，比如ControlDict.toml文件的字典含义，参数设置等
4. **程序手册**：程序各个部分的内容，比如程序结构，并行原理等
5. **算例手册**：针对典型算例，进行介绍，比如参数、模型与运行
6. **API文档**：程序中各个函数与类的调用接口
7. **其它资料**：其它有助于使用程序的资料，比如toml语法，C++语法等
